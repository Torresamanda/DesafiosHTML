Cascading Style Sheet

Style - CSS
É usado para formatar o layout de uma página web.

O CSS pode ser utilizado no documento de 3 formas:
Inline - Como o nome já diz é utilizado na linha de código utilizando o atributo "style" dentro do elemento HTML
Interno - Utilizando o atributo "style" dentro do elemento "head"
Externo - É utilizado usando um elemento "link" para vincular a um arquivo CSS externo

De toda forma o mais comum é utizar o arquivo CSS externo.

Algumas propriedades CSS mais usadas:
Color- Define a cor do texto a ser usado
            h1 {
                color: blue;
            }
font-family- Define a fonte a ser usado
            h1 {
                font-family: verdana;
            }
font-size- Define o tamanho da fonte a ser usado
            h1 {
                font-size: 300%;
            }
Border- Define uma borda em torno de um elemento HTML
        p {
            border: 2px solid powderblue;
          }
Padding- Define um preenchimento entre o texto e a borda
            p {
                padding: 30px;
              }
Margin- Define um espaço fora da borda
            p {
                margin: 50px;
              }
